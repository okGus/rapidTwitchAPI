### About
Unofficial Twitch API Web Scraper

## Four endpoints

/Homepage
- Live channels in homepage (small footprint)
- Url
- Channel name

/Categories
- Most popular categories by views
- Link
- Title
- Views

/Category/:catogoryId
- Live channels within a category
- Channel Name
- Title
- View count
- Link

/Channel/:chanelId
- Info about channel
- If they are live or not
- View count
- Title
- Category
- Time alive
